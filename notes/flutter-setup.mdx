## Set up

Dowload flutter

Download [command-line tool](https://developer.android.com/studio#command-line-tools-only)

Download [platform-tools](https://developer.android.com/tools/releases/platform-tools) and place in../android-sdk 

Use command-line tool **sdkmanager** to download **platforms** and **build-tools**

flutter config --android-sdk
